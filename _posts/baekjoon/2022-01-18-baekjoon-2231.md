<h3>문제 풀이</h3>
생성자를 찾고, 그 중 가장 작은 생성자를 골라 출력하는 문제이다. 단순하게 생각해서 1부터 올라가면서 그 수의 분해합을 구하고, 이 값을 우리의 목표값이랑 비교하여 같으면 가장 작은 생성자를 찾은 것이다. 코드는 다음과 같다.

```python
n = int(input())

def sum(n):
    a = []
    for i in str(n):
        a.append(i)
    sum = n
    for i in a:
        j = int(i)
        sum+=j
    return sum

def main():
    for i in range(n):
        if sum(i)==n:
            print(i)
            return 0
    print("0")
    return 0

main()
```
