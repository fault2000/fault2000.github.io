<h3>문제 풀이</h3>

재귀 문제이다. 피보나치의 정의에 충실하게 코드를 적으면 다음과 같다.

```python
def fibo(n):
    if n==0:
        return 0
    elif n==1:
        return 1
    else:
        return fibo(n-2)+fibo(n-1)

a = int(input())
print(fibo(a))
```
